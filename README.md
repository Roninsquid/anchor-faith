# ⚓ Anchor

> *A place to begin again.*

Anchor is a quiet place for people seeking peace, Scripture, reflection, and prayer.

It is not built to maximize attention.

It is built to help people return their attention to God.

---

## Our Mission

Anchor exists to gently guide people toward God's Word through thoughtful design, compassionate language, and meaningful reflection.

Every screen, every sentence, and every feature should help create a peaceful experience for someone who may be carrying something heavy.

---

## What Makes Anchor Different

Anchor is intentionally designed to be calm.

We believe less can be more.

Instead of endless notifications and distractions, Anchor invites people to slow down.

Instead of trying to have every answer, Anchor begins with Scripture.

Instead of rushing people forward, Anchor creates space to pause, reflect, pray, and begin again.

---

## Current Features

- 🌅 Time-based welcome
- ⚓ Peaceful landing experience
- 📖 Guided Scripture reveal
- 💭 Quiet moments for reflection
- 🙏 Foundation for guided prayer
- 📚 The First Twelve journey framework

---

## Project Philosophy

Every feature should answer one question:

> **If someone arrived here carrying something heavy, what is the kindest next step?**

If a feature doesn't help answer that question, it probably doesn't belong.

---

## Project Documents

- **VISION.md** — Why Anchor exists.
- **FIRST_TWELVE.md** — The original journeys that define Anchor's voice.

These documents are considered just as important as the code itself.

---

## Development Principles

- Every commit leaves Anchor working.
- One thoughtful decision at a time.
- Protect the mission before adding features.
- Make it feel peaceful before making it feel impressive.
- If the code becomes confusing, rewrite it cleanly.

---

## Roadmap

### Phase I — Welcome ✅

Create a peaceful sanctuary where visitors can pause before engaging.

### Phase II — Listen 🚧

Build the first twelve handcrafted journeys that respond with Scripture, reflection, prayer, and one gentle next step.

### Phase III — Guide

Thoughtfully introduce intelligent guidance while keeping Scripture at the center.
